# Segment 구조 및 역할

슬라이스 **내부** segment다. slice 경계와 혼동하지 않는다 ([layer-rules.md](./layer-rules.md)).

## `ui`

- **역할**: UI 표시·표현.
- **포함**: 컴포넌트, 포맷터, 스타일 등.

## `api`

- **역할**: 백엔드와의 통신.
- **포함**: 요청 함수, (쿼리 옵션 등) 데이터 타입, React Query 설정, 매퍼.

## `model`

- **역할**: 데이터 구조·상태·도메인 규칙.
- **포함**: 스키마, 인터페이스, 스토어, 비즈니스 로직.

## `lib`

- **역할**: **해당 슬라이스 내부** 보조 코드.
- **포함**: 슬라이스 전용 헬퍼·유틸.
- **주의**: 여러 슬라이스에서 쓰이면 `shared/lib`로 이동.

## `config`

- **역할**: 슬라이스별 설정.
- **포함**: 환경별 설정, feature flag, 상수.
